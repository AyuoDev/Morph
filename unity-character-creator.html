How to Use Morphara Characters in Unity

  1. Create your character in Morphara (takes 2-5 minutes)
    • Choose base mesh
    • Customize face with blend shapes
    • Add clothing and accessories
    • Paint custom textures
    • Apply animations
  2. Export as FBX
    • Click "Export" button
    • Choose "FBX" format
    • Download ZIP file (contains FBX + textures)
  3. Import into Unity
    • Drag the FBX file into your Unity Assets folder
    • Drag texture files into Materials folder
    • Assign textures to character materials
    • Add to scene - ready to use!

ðŸ’¡ Pro Tip: Use Unity's Animator Controller to blend between Morphara animations. All animation clips are automatically imported and ready to use.

Frequently Asked Questions

Can I use Morphara characters in commercial Unity games?

Yes! Premium and Studio plans include commercial usage rights. Free plan exports are for personal/educational use only.

Do the animations work with Unity's Animator?

Absolutely. All animations are baked into the FBX and work seamlessly with Unity's Animator Controller. You can blend between animations just like any Unity humanoid character.

Are the characters optimized for mobile?

Yes. Our characters have game-ready polygon counts suitable for mobile platforms. Low-poly base meshes are specifically designed for mobile performance.

Can I modify the characters after importing to Unity?

Yes! The FBX format is fully editable in Unity. You can adjust materials, add your own textures, modify blend shapes, and customize as needed.

Do I need to install anything?

No! Morphara runs entirely in your browser. No downloads, no plugins, no software installation required. Just open the site and start creating.
